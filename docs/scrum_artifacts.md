## Scrum Artifacts

1. **Product Backlog:** GitHub Projects board with user stories
2. **Sprint Backlogs:** Decomposed tasks in project board + Slack
3. **Increment:** Working software after each sprint
4. **Definition of Done:**
    - PR requirements met (set up by chat team)
    - Tests passing (RSpec + Cucumber)
    - Coverage >90%
    - Code reviewed via PR
    - Merged to dev branch
    - Documentation updated
